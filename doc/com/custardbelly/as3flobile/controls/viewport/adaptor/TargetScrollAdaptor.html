Classpublic class TargetScrollAdaptor
InheritanceTargetScrollAdaptor Object
Implements ITargetScrollAdaptor

TargetScrollAdaptor assumes scrolling animation of a target display based on a target position. Typically this is used by a IScrollViewportStrategy to move content along the axes toward a target. IScrollViewportStrategy mainly handles a 'loose' scrolling based on user gesture and is not focused on finding a specific position. This hopes to alleviate that responsibility.
