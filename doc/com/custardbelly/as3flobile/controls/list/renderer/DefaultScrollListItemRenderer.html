Classpublic class DefaultScrollListItemRenderer
InheritanceDefaultScrollListItemRenderer flash.display.Sprite
Implements IScrollListItemRenderer

DefaultScrollListItemRenderer is the default item renderer for a IScrollListContainer. The recognize object model for the data supplied to this class is {label:"your label"}. The label property of the object model is represented textually.
